## 🚀 DÉMARRAGE IMMÉDIAT (3 ÉTAPES)

### 1. Installation (30 secondes)
```bash
# Windows
SETUP.bat

# Mac/Linux
bash SETUP.sh
```

### 2. Démarrage Backend (Terminal 1)
```bash
cd backend
npm start
```
Attendez de voir: `🚀 Serveur Ti'Coeur démarré sur http://localhost:5000`

### 3. Démarrage Frontend (Terminal 2)
```bash
cd frontend
npm run dev
```
Attendez de voir: `➜ Local: http://localhost:5173/`

### 4. Ouvrir le Navigateur
Allez sur: **http://localhost:5173**

🎉 **VOILÀ! Votre site est LIVE!**
