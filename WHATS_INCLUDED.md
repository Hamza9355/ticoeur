## 🚀 Comment Démarrer

### Option 1: Script automatique (Recommandé)
```bash
# Windows
SETUP.bat

# macOS/Linux
bash SETUP.sh
```

### Option 2: Manuel
```bash
# Terminal 1 - Backend
cd backend
npm install
npm start

# Terminal 2 - Frontend
cd frontend
npm install
npm run dev
```

Puis ouvrez: **http://localhost:5173**
